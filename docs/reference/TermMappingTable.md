# Ellmer type schema for a table of ontology term mappings

<div class="ref-description section level2">

An array of `TermMapping` objects, suitable for passing as the `type`
argument to `chat$chat_structured()` when multiple concepts are to be
mapped in a single call.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
TermMappingTable
```

</div>

</div>

<div class="section level2">

## Format

An ellmer `TypeArray` whose items conform to `TermMapping`.

</div>
